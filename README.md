# Coffee-Sales-Analysis

# ☕ Coffee Shop Sales Analysis Dashboard

An end-to-end Excel Data Analytics project built using the Maven Analytics Coffee Shop Sales dataset.

---

## 📌 Project Overview

This project analyzes more than **149,000 coffee shop transactions** to uncover business insights related to sales performance, customer behavior, store performance, and product trends.

The dashboard was developed entirely in Microsoft Excel using Power Query, Power Pivot, DAX, Pivot Tables, and interactive visualizations.
